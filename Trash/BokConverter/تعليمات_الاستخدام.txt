# تعليمات استخدام برنامج تحويل ملفات .bok إلى .accdb

## ✅ تم اختبار البرنامج بنجاح!

تم تشغيل البرنامج وتحويل 3 ملفات بنجاح:
- الاختيار.bok → الاختيار.accdb
- بغية السائل.bok → بغية السائل.accdb  
- عمدة.bok → عمدة.accdb

## طرق التشغيل

### 1. الطريقة الأسهل (ملف Batch):
اضغط مرتين على الملف: `تشغيل_البرنامج.bat`

### 2. الطريقة اليدوية (PowerShell):
```
cd "d:\test5\BokConverter"
powershell.exe -ExecutionPolicy Bypass -File "BokConverter.ps1"
```

## الملفات المتوفرة

### للاستخدام المباشر:
- `تشغيل_البرنامج.bat` - ملف التشغيل السريع
- `BokConverter.ps1` - البرنامج الرئيسي

### لمطوري C#:
- `BokConverter.csproj` - ملف المشروع
- `Program.cs` - الكود المصدري بـ C#

### التوثيق:
- `README.md` - دليل شامل
- `تعليمات_الاستخدام.txt` - هذا الملف

## المتطلبات
- ✅ Microsoft Access (مثبت ويعمل)
- ✅ Windows PowerShell
- ✅ صلاحيات القراءة والكتابة في المجلدات

## ملاحظات مهمة
- البرنامج لا يحذف الملفات الأصلية (.bok)
- إذا كان الملف المحول موجود مسبقاً، سيتم تجاهله
- الملفات المحولة تُحفظ في مجلد "access file"
- يستخدم مجلد مؤقت أثناء التحويل ثم ينظفه

## نتائج الاختبار الأخير
```
=== BOK to ACCDB Converter ===
Source folder: d:\test5\bok file
Output folder: d:\test5\access file

- Successfully converted: 3 files
- Skipped (already exists): 1 files  
- Failed to convert: 0 files
- Total files: 4 files
```

## استكشاف الأخطاء
إذا ظهرت أخطاء:
1. تأكد من إغلاق جميع ملفات Access
2. تأكد من وجود صلاحيات كتابة في المجلدات
3. تأكد من سلامة ملفات .bok الأصلية
4. شغل PowerShell كمسؤول إذا لزم الأمر
